# CISP Project 24- Word Guessing Game
## Overview
Word guessing game written in Java.
Sselects a random word from a local word list file.
Guess one letter at a time to slowly reveal the hidden word.

## Program details
* Random word is selected at startup
* Word is hidden
* You enter single letter guesses
* Correct guesses reveal letters
* Incorrect guesses reduce attempts
* Game ends when word is solved or attempts reach zero

## Features

* Random word selection from file
* Letter-by-letter guessing system
* Limited number of guesses
* Win and lose conditions
* Cheat mode option to show answer
* Case-insensitive word handling
  
## Project Structure
* models: Word data storage
* viewers: User interface and game loop
* controllers: Game logic and random word selection

## Key Ideas
* MVC Structure
* Arrayss
* Game loop logic
